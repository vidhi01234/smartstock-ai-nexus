# 🚀 Smart Inventory Forecast & Alert System

An AI-powered full-stack application that helps businesses predict demand, manage inventory efficiently, and avoid stockouts or overstocking.

---

## 📌 Features

* 📊 **Inventory Dashboard** – Real-time view of stock levels
* 🤖 **Demand Forecasting** – ML-based prediction using historical data
* 🔔 **Smart Alerts** – Get notified for low stock or overstock
* 📈 **Cost Intelligence** – Analyze inventory costs and trends
* 🔐 **Authentication System** – Secure login for users

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/vidhi01234/smartstock-ai-nexus.git
cd AIDemand
```

---

### 2️⃣ Backend Setup

```
cd backend
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
python app.py
```

---

### 3️⃣ Frontend Setup

```
cd frontendAI
npm install
npm run dev
```

---

## 🧠 How It Works

1. User inputs inventory data
2. Backend processes data using ML models
3. Predictions are generated
4. Frontend displays insights and alerts
